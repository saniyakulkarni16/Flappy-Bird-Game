🐤 Flappy Bird Game – HTML, CSS & JavaScript

A simple yet addictive Flappy Bird clone built using vanilla HTML, CSS, and JavaScript. In this game, you guide a tiny bird through a series of obstacles with just one control — a single tap to flap you're flying between pipes — testing your reflexes, rhythm, and patience!

🎮 Features:

1. Smooth, tap-to-flap bird mechanics

2. Moving pipe obstacles with collision detection

3. Score tracking system

4. Retro-styled design using pure CSS

5. Fully responsive for desktop and mobile

🚀 Tech Stack:

HTML – Structure and layout

CSS – Styling and animations

JavaScript – Game logic and interactivity
